## 简介

把 php 项目使用 composer 管理起来，让其管理依赖库和自动加载，所以选择了一部分基本常用库并且发布到 [packagist](https://packagist.org), 以后只需要引入这个基本库就够了

## 使用

```
composer require zhu2688/framework
```
